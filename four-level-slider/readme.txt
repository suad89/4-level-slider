Slider plugin with 4 images. It has Title, Link and Image. Just save your changes and copy shortcode where you want to place this slider.

== Description ==
This is very simple slider plugin with 4 images. Add Title, Link and Image then hit save changes. Finally copy this shortcode <b>[four_level_slider]</b> and paste it where you want to show this slider. For best experience upload images in size 1920px x 1282px.

1. Image upload, connected with wordpress media
2. Easy to put anywhere you want
3. You can use it like navigator, just add links to your pages, title of page and image.
4. You can change colors

== Installation ==
* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings and fill settings.

== Frequently Asked Questions ==
= What is the purpose of this plugin =
Sometimes you need list of important things so here you can show it in interesting and different way.

= Can I change a colors =
You can change background color, by default it has opacity of 0.85 so have that in mind. Color of text and circles also can be changed.

= Can I add images in different format =
No! Please add the images in dimension 1920px x 1282px, and it will fit perfectly.

= Can I use it in full screen =
Yes you can but it is recommended for using in bootstrap container which is 1200px width.
